## 〇、 伯特兰...?

乔瑟夫·伯特兰 （Bertrand Louis François Joseph, 1822-1900）是19世纪法国的数学家。约17岁获得博士学位。他的研究领域广泛， 弄出了另外三个“伯特兰问题”，如下：

- 最著名的，至今未定：**概率论中的伯特兰悖论**（Bertrand's paradox, in ***Calcul des probabilités***， 1889） 考虑一个内接于圆的等边三角形，随机选择圆的一条弦， 求这条弦等于该三角形边长的概率）
- 经济学中的伯特兰悖论（Bertrand paradox, 描述了一种情况：两个参与者（企业）达到纳什均衡状态，此时两家企业都收取等于边际成本（“MC”）的价格。）
- 贝叶斯概率论中与蒙提霍尔问题等价的伯特兰盒子悖论（Bertrand's Box Paradox, equivalent to the Monty Hall problem）

> 1845年，伯特兰猜想：对于任意 $n\geq 1$，在  $n$  与 $2n$ 之间必存在至少一个素数 $p$ 。通过素数表，他验证了 $ n<3\times10^6$ 的情形。
> 1850年，切比雪夫首次给出分析证明， 把猜想变成了“公设”。半个多世纪后，拉马努金（Ramanujan, 1919）与厄多斯（Erdős, 1932）—— 分别用比较奇怪的算法和初等方法给出了证明。

## 一、问题的等价转换

伯特兰假设：
$$ \forall n \in \mathbb{Z}_{\ge 2}\, \exists p \in \mathbb{P}: n \le p < 2n $$

<!-- 【结构】紧接此公式块的下一行以孤立全角句号「。要证明它…」开头，是悬挂标点。请把句号删去，使该段落自然衔接公式。 -->

。要证明它，可以直接数素数的个数，两位数学家不约而同地转向了一个“替身”——中心二项式系数 

$$\binom{2n}{n} = \frac{(2n)!}{(n!)(2n-n)!} = \frac{1\times 2\times \cdots \times 2n}{(1\times 2\times \cdots \times n)^2}.
$$


为什么呢？考虑它的两个基本性质：

1. **下界**：因为 \(\binom{2n}{n}\) 是 \((1+1)^{2n}=4^n\) 展开式中最大的一项，且共有 \(2n+1\) 项，故有
   \[
   \frac{4^n}{2n+1} \le \binom{2n}{n} \le 4^n.
   \]
   左边的不等式告诉我们 \(\binom{2n}{n}\) 随 \(n\) 增长得很快——与 \(4^n\) 同阶。
2. **素数分解**：\(\binom{2n}{n}\) 的分子分母都由连续整数相乘而成，其素数分解可以通过一个经典公式精确刻画：
   \[
   o_p(n!) = \sum_{j=1}^\infty \left\lfloor \frac{n}{p^j} \right\rfloor,
   \]
   其中 \(o_p(m)\) 表示 \(m\) 中素数 \(p\) 的指数。

如果区间 \((n,2n]\) 中没有素数，那么 \(\binom{2n}{n}\) 的所有素因子都只能来自不超过 \(n\) 的数。这会给 \(\binom{2n}{n}\) 的大小加上一个“天花板”。当天花板低于其应有的“地板”（即下界 \(4^n/(2n+1)\)）时，矛盾便出现了。这就是两个证明共同的逻辑骨架。

## 二、拉马努金的解析：引入 \(\theta(x)\) 与 \(\psi(x)\)

拉马努金的证明更富于分析色彩。他引入了两个由切比雪夫首创的函数，将离散的素数计数问题连续化。

### 2.1 函数定义

对于实数 \(x\ge1\)，定义：

\[
\theta(x) = \sum_{p\le x} \log p, \qquad
\psi(x) = \sum_{m=1}^\infty \theta\!\left(x^{1/m}\right).
\]

这里 \(p\) 取遍所有素数。\(\theta(x)\) 直接对所有不超过 \(x\) 的素数取对数求和，因此 \(\theta(x)\) 的增长直接反映了素数的累积。伯特兰假设等价于：

\[
\theta(2x) - \theta(x) > 0 \quad (\forall x\ge1),
\]

因为一旦差值大于0，就必然有至少一个素数落在 \((x,2x]\) 中，其对数值贡献了这个差值。

\(\psi(x)\) 则是 \(\theta\) 在所有根号层次上的叠加。注意到当 \(x<2\) 时，\(x^{1/m}<2\)，除了 \(p=2\) 之外没有素数，因此求和实际上是有限项。\(\psi\) 的优点在于它与阶乘的对数有极简洁的关系。

<!-- 【核对】此句措辞含混：x<2 时 θ(x)=0，并不存在任何素数（最小素数为 2），「除了 p=2 之外没有素数」易误导。建议改为：「对固定的 x，当 m 足够大时 x^{1/m}<2，从而 θ(x^{1/m})=0，故 ψ(x) 的求和只有有限多项非零。」 -->

### 2.2 与阶乘的联系

利用 \(n!\) 的素数分解公式，拉马努金证明了（引理2）：

\[
\log \lfloor x \rfloor! = \sum_{\ell=1}^\infty \psi\!\left(\frac{x}{\ell}\right).
\]

由此可以构造交错级数，得到关键的恒等式：

\[
\log \lfloor x \rfloor! - 2\log \lfloor x/2 \rfloor! = \sum_{\ell=1}^\infty (-1)^{\ell-1} \psi\!\left(\frac{x}{\ell}\right). \tag{5}
\]

由于 \(\theta\) 和 \(\psi\) 都是单调不减函数，从定义和交错级数的性质可推出两组核心不等式：

\[
\psi(x) - 2\psi(\sqrt{x}) \le \theta(x) \le \psi(x), \tag{6}
\]
\[
\psi(x) - \psi(x/2) \le \log \lfloor x \rfloor! - 2\log \lfloor x/2 \rfloor! \le \psi(x) - \psi(x/2) + \psi(x/3). \tag{7}
\]

### 2.3 核心引理：对数阶乘差的界

拉马努金证明中最具技巧的一步，是给出 \(\log \lfloor x \rfloor! - 2\log \lfloor x/2 \rfloor!\) 的上下界（引理3）：

\[
\frac{2}{3}x < \log \lfloor x \rfloor! - 2\log \lfloor x/2 \rfloor! < \frac{3}{4}x.
\]

该引理本质上来源于对 \(\binom{2n}{n}\) 的上下界估计，并巧妙地处理了取整函数带来的波动。它避开了斯特林公式，使整个证明保持初等。

结合(7)式，立刻得到关于 \(\psi\) 的不等式：

\[
\psi(x) - \psi(x/2) < \frac{3}{4}x \quad (x>0), \qquad
\psi(x) - \psi(x/2) + \psi(x/3) > \frac{2}{3}x \quad (x\ge300). \tag{17}
\]

### 2.4 导出矛盾

将(17)的第一式反复迭代（令 \(x \to x/2^k\) 并求和），可得：

\[
\psi(x) < \frac{3}{2}x \quad (\text{原文实际得到更紧的 }\psi(x)<\frac{3}{4}x, \text{足以使用}).
\]

<!-- 【核对】此括注疑似错误：由 ψ(x)−ψ(x/2)<3/4 x 反复迭代求和只能得 ψ(x)<3/4·2x=3/2 x，不可能得到更紧的 3/4 x（那比丢掉负项 −ψ(x/2) 还紧，自相矛盾）。文献中拉马努金原始上界是 ψ(x)<2x·ln2≈1.386x，对应本文归一化即 3/2 x。建议删去括注，或改为「原文用自然对数得 ψ(x)<2x ln2≈1.386x，本文归一化后为 3/2 x，已足够使用」。 -->

再利用(6)与(17)的第二式，经过放缩可得对于 \(x>300\)，

<!-- 【核对】请确认 (21) 式常数：本文用 2/3、3/4 归一化得 θ(x)−θ(x/2)>x/6−3√x；文献（自然对数版，Meher–Murty）为 θ(x)−θ(x/2)>(x/3)ln2−ln√(2x)−2√x·ln2。两者结构一致、阈值 324=18² 自洽，但系数 1/6 与 3√x 的具体来历建议补一行推导说明，便于读者复核。 -->

\[
\theta(x) - \theta(x/2) > \frac{x}{6} - 3\sqrt{x}. \tag{21}
\]

当 \(x\ge324\) 时，右端非负，故 \(\theta(x)-\theta(x/2)>0\)，即区间 \((x/2,x]\) 内有素数。代换 \(y=x/2\)，即知对 \(y\ge162\)，伯特兰假设成立。小值验证后，定理得证。

拉马努金的方法还顺带给出了素数个数的下界：

\[
\pi(x) - \pi(x/2) > \frac{1}{\log x}\left(\frac{x}{6} - 3\sqrt{x}\right),
\]

由此可知区间内的素数个数随 \(x\) 无限增长——这已触及素数定理的初等形态。

## 三、厄多斯：直接放缩素数分解

1932年，19岁的厄多斯发表了他人生的第一篇论文，给出了一个全然不同的初等证明。它不使用任何连续函数，仅依赖整除性分析与最朴素的不等式。

### 3.1 假设无素数时的分解限制

假设对于某个 \(n\ge3\)，区间 \((n,2n]\) 中没有素数。考察 \(\binom{2n}{n}\) 的素数分解。厄多斯首先注意到一个简单事实：

> 若 \(\frac{2}{3}n < p \le n\)，则 \(p\) 不整除 \(\binom{2n}{n}\)。

原因在于：\(p\) 在 \((2n)!\) 中出现2次（\(p\) 和 \(2p\)），在 \((n!)^2\) 中各出现1次，指数相减为0。

因此，如果 \((n,2n]\) 无素数，则 \(\binom{2n}{n}\) 的所有素因子都 \(\le \frac{2n}{3}\)。

### 3.2 控制每个素因子的指数

对任意素数 \(p\)，它在 \(\binom{2n}{n}\) 中的指数满足：

\[
o_p\!\left(\binom{2n}{n}\right) = \sum_{j=1}^{r(p)} \left( \left\lfloor \frac{2n}{p^j} \right\rfloor - 2\left\lfloor \frac{n}{p^j} \right\rfloor \right) \le r(p),
\]

其中 \(p^{r(p)}\le 2n < p^{r(p)+1}\)。由此得 **\(p^{o_p}\le 2n\)**。

这意味着：

- 若 \(p > \sqrt{2n}\)，则 \(r(p)=1\)，故指数至多为1；
- 任何素因子的贡献不超过 \(2n\)。

### 3.3 拼凑上界并引发矛盾

于是 \(\binom{2n}{n}\) 可被如下放大：

\[
\binom{2n}{n} \le \underbrace{(2n)^{\sqrt{2n}}}_{\text{小素数的最大可能贡献}} \times \underbrace{\prod_{\sqrt{2n} < p \le 2n/3} p}_{\text{大素数各贡献一次}}.
\]

再利用一个初等引理——**对所有 \(n\ge2\)，\(\prod_{p\le n} p \le 4^n\)**（可用归纳法证明），得到：

\[
\binom{2n}{n} \le (2n)^{\sqrt{2n}} \cdot 4^{2n/3}.
\]

结合下界 \(\binom{2n}{n} \ge \frac{4^n}{2n+1}\)，我们要求：

\[
\frac{4^n}{2n+1} \le (2n)^{\sqrt{2n}} 4^{2n/3}.
\]

然而，当 \(n\) 足够大时（\(n\ge468\)），这个不等式必然失效，因为左边以 \(4^n\) 的量级增长，右边仅是约 \(4^{2n/3}\) 乘一个次指数因子。矛盾说明假设错误，即对 \(n\ge468\)，\((n,2n]\) 中必有素数。小值直接枚举验证即可。

## 四、双星辉映：两种证明的对比与启示

| 维度                 | 拉马努金 (1919)                                      | 厄多斯 (1932)                            |
| -------------------- | ---------------------------------------------------- | ---------------------------------------- |
| **核心工具**   | 切比雪夫函数\(\theta,\psi\)，对数阶乘的恒等式        | \(\binom{2n}{n}\) 的素数分解与初等不等式 |
| **关键中间量** | \(\log\lfloor x\rfloor! - 2\log\lfloor x/2\rfloor!\) | \(\binom{2n}{n}\) 本身的上下界           |
| **技术难点**   | 引理3的上下界证明（避开斯特林公式）                  | 处理小素数和大素数的不同指数规律         |
| **证明长度**   | 较长，涉及函数构造与级数运算                         | 极短，核心段落不过一页                   |
| **推广潜力**   | 直接导出素数个数下界，具有解析深度                   | 可延展至素数定理初等证明的思路启发       |


### 4.1 我们能推断什么？

1. **同一源泉的不同流向**：两个证明都起源于 \(\binom{2n}{n}\) 的素数分解，这暗示了二项式系数在素数分布研究中的核心地位。
2. **初等方法的惊人力量**：在人们普遍认为素数定理必须依赖复分析的时代，这两个证明（尤其是厄多斯证明）展示了初等组合方法完全可以攻克非平凡的素数分布难题。它们为后来塞尔伯格和厄多斯本人初等证明素数定理埋下了伏笔。
3. **风格的互补性**：拉马努金的证明带有强烈的19世纪分析传统色彩，适合喜欢函数恒等式与渐近估计的读者；厄多斯的证明则是20世纪组合数学“从定义出发”的典范，更适合作为本科教学案例。
4. **历史地位的差异**：厄多斯的证明因其极致的简洁被收录于《Proofs from THE BOOK》（天书证明），成为数学美的象征。而拉马努金的证明虽稍早且同样精彩，却因篇幅与技巧的繁复而较少被大众所知——这正是我们重拾它的价值所在。

## 五、结语：素数间隔的永恒谜题

伯特兰假设只是素数分布研究的起点。它告诉我们 \(n\) 与 \(2n\) 之间必有素数，更揭示了素数出现的频率远高于随意猜测。沿着先人开辟的道路，后人不断缩小区间长度：从 \((n,2n]\) 到 \((n,n+n^\theta)\)，再到黎曼猜想所预言的 \((n,n+\sqrt{n}\log n]\)......
以及至今仍未解决的，看似简单的勒让德猜想—— $n^2$  与 \((n+1)^2\) 之间是否必有素数。
